# Changelog

## v1.1.0 — Astrocartography & Local Space

**Новые возможности:**
- **Астрокартография (ACG):** Добавлены расчеты проекций планетных линий на географическую карту.
- **Локальное пространство (Local Space):** Реализованы расчеты азимутов и высот планет для проекции гороскопа на локальное окружение.
- **Фильтрация аспектов:** Добавлен гибкий фильтр для отсеивания незначительных аспектов и вывода только ключевых (мажорных, минорных и т.д.) в зависимости от запроса.

**Улучшения:**
- Оптимизация расчетов движка для поддержки новых модулей пространства.

## v1.0.0 — Initial Release

Первый публичный релиз изолированного сервиса астрологических расчетов.

**Основные возможности:**
- **Натальные карты (`/natal`):** Расчёт положений планет, домов и аспектов.
- **Транзиты (`/predict/daily`):** Расчёт транзитных планет и аспектов к наталу на конкретную дату.
- **Синастрия (`/synastry`):** Наложение двух карт, расчёт аспектов между партнёрами и попадания планет в дома друг друга.
- **Хорарная астрология (`/horary`):**
  - Поддержка точного UTC-времени (`dt_utc`).
  - Автоматический расчёт статусов планет (сожжение, ретроградность).
  - Определение Луны без курса (Void of Course) и других важных факторов.

**Технические детали:**
- **Стек:** Python 3.11, FastAPI, Uvicorn.
- **Ядро:** Kerykeion + Pyswisseph (GPL-лицензия изолирована внутри этого сервиса).
- **Безопасность:** Проверка заголовка `X-API-Key` (синхронизирован с `INTERNAL_API_KEY`).
- **Docker:** Оптимизированный образ для быстрой сборки и деплоя.
